# QoL Alteryx-ODI

Ferramenta Python para parsing, conversao e manipulacao de workflows Alteryx (.yxmd) e packages ODI em formato XML.

## Descricao

Utilitario com interface grafica (Tkinter) para:
- Parsing de workflows Alteryx XML
- Parsing de packages ODI XML
- Conversao entre formatos Alteryx e ODI
- Processamento em lote de multiplos arquivos
- Validacao de regras de workflow
- Visualizacao de diffs entre arquivos XML

## Instalacao

```bash
# Linux
chmod +x install.sh
./install.sh
source venv/bin/activate
python main.py
```

```batch
REM Windows
setup.bat
```
